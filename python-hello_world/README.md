# Python - Hello, World

This project is part of the curriculum at Holberton School. It focuses on learning the basics of Python programming, including using the interpreter, printing, working with strings and numbers, and understanding Python style conventions.

## üìÅ Project Structure

Each file implements a specific task related to Python scripting fundamentals.

### Files

- `2-print.py`  
  Prints a message using the `print` function.

- `3-print_number.py`  
  Prints an integer variable using an f-string without casting.

- `4-print_float.py`  
  Prints a float variable with two decimal places using f-strings.

- `5-print_string.py`  
  Prints a string three times and then prints the first 9 characters of that string.

- `6-concat.py`  
  Concatenates two strings using variables and prints a complete sentence.

- `7-edges.py`  
  Extracts and prints parts of a string: first 3 letters, last 2 letters, and everything in between.

- `8-concat_edges.py`  
  Creates and prints a sentence by slicing and concatenating parts of a string without using new variables or literals.

- `9-easter_egg.py`  
  Prints "The Zen of Python" by Tim Peters.
