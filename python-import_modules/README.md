# Python - Import & Modules

This project is part of the curriculum at Holberton School. It focuses on learning how to import functions, use modules, and work with command-line arguments in Python.

## üìÅ Project Structure

Each file implements a specific task related to Python imports, modules, and basic scripting operations.

### Files

- `0-add.py`  
  Imports the `add` function from `add_0.py` and prints the result of `1 + 2`.

- `1-calculation.py`  
  Imports functions from `calculator_1.py` and performs basic arithmetic operations (addition, subtraction, multiplication, division).

- `2-args.py`  
  Prints the number of command-line arguments and their values.

- `3-infinite_add.py`  
  Prints the result of the addition of all command-line arguments.

- `4-hidden_discovery.py`  
  Prints all names defined in the compiled module `hidden_4.pyc` (located on the sandbox).

- `5-variable_load.py`  
  Imports and prints the value of the variable `a` from `variable_load_5.py`.
